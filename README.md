# Installation & Setup
**Prerequisites**
- Python 3.8+
- Google AI API key
- Required packages (see requirements.txt)
  
**Quick Start**
- Clone the repository
- Install dependencies: pip install -r requirements.txt
- Add your Google AI API key through the web interface
- Run: python main.py
- Access at http://localhost:7867
